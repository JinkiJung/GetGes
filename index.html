Welcome to GetGes page.

This framework provides an introductory steps for developing gesture recognition module using the Myo Armband. GetGes supports both Windows and macOS environment, which are represented with Microsoft's Visual Studio and Xcode respectively. With an OpenCV3.0 support, you can utilize all machine learning modules in GetGes project. Also GetGes is able to handle multiple Myos, as you can see the sensory data graph in the individual window.

Developing Environment

Windows

  • Windows 10
  • Microsoft's Visual Studio 2013

macOS

  • macOS Sierra 10.12
  • Xcode 7.3

common

  • Myo Connect 1.0.1
  • OpenCV 3.0.0
